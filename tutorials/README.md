# Tutorials

Step-by-step guides for building with Bakend.

## Prerequisites

- [Bun](https://bun.sh) 1.1+ (development) **or** the `bak` binary from [GitHub Releases](https://github.com/alpbak/bakend/releases)
- Git
- `curl` for API requests
- Port `8080` available locally

## Tutorials

| # | Tutorial | What you'll learn |
|---|----------|-------------------|
| 01 | [Build a Todo API](01-todo-api.md) | Collections, auth, CRUD, owner permissions, functions |
| 02 | [Add Realtime](02-realtime-app.md) | WebSocket subscriptions and live updates |
| 03 | [Deploy to a VPS](03-deploy-vps.md) | Binary install, systemd, health checks |
