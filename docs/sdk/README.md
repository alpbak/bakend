# SDKs

Client libraries for applications connecting to Bakend.

| SDK | Package | Install |
|-----|---------|---------|
| JavaScript / TypeScript | `@bakend/client` | `npm install @bakend/client` |
| Dart / Flutter | `bakend` | `dart pub add bakend` |
