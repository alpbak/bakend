# SDK

Bakend provides client libraries for JavaScript/TypeScript and Dart.

## Install

**npm:**

```bash
npm install @bakend/client
```

**pub.dev:**

```bash
dart pub add bakend
```

**Monorepo development:**

```bash
bun install   # resolves @bakend/client from sdk/javascript
```
